# تويتر عراقي 🇮🇶

نسخة مصغّرة من تويتر مبنية بـ PHP خام (بدون فريموورك) + MySQL، بتصميم عربي (RTL) مستوحى من تويتر.

تم اختبار كل الميزات فعلياً (تسجيل، نشر، لايك، كومنت، متابعة، رسائل، إشعارات، بحث، رفع صور) وتشتغل 100%.

## الميزات المتوفرة

- تسجيل حساب / تسجيل دخول (كلمات السر مشفّرة بـ bcrypt)
- نشر تغريدات (نص + صورة)
- لايك بدون تحديث الصفحة (AJAX)
- تعليقات على التغريدات
- متابعة / إلغاء متابعة
- صفحة بروفايل (صورة شخصية + غلاف + نبذة) قابلة للتعديل
- رسائل خاصة بين المستخدمين
- إشعارات (لايك، تعليق، متابعة)
- بحث عن مستخدمين
- صفحة استكشاف لكل التغريدات
- تصميم متجاوب (Responsive) يشتغل بالموبايل

## المتطلبات

- PHP 8.0 أو أحدث (مع امتداد PDO و pdo_mysql)
- MySQL أو MariaDB
- سيرفر ويب (Apache/Nginx) أو تگدر تشغّله مباشرة بسيرفر PHP المدمج

## طريقة التشغيل (أسرع طريقة للتجربة)

### 1. جهّز قاعدة البيانات

```bash
mysql -u root -p < database/database.sql
```

هذا الأمر راح ينشئ قاعدة بيانات اسمها `twitter_iraqi` ويسوي جدول تجريبي فيه مستخدم واحد:
- **اسم المستخدم:** ali
- **كلمة السر:** 123456

### 2. جهّز ملف الإعدادات

انسخ `.env.example` وسميه `.env`، وعدّل بيانات الاتصال حسب سيرفرك:

```bash
cp .env.example .env
```

```
DB_HOST=127.0.0.1
DB_NAME=twitter_iraqi
DB_USER=root
DB_PASS=كلمة_سر_قاعدة_بياناتك
APP_URL=http://localhost:8000
```

> ملاحظة: إذا صارت مشكلة "Access denied" وأنت مستخدم root بدون باسوورد، سوّي مستخدم جديد بقاعدة البيانات وخله يتصل بـ TCP:
> ```sql
> CREATE USER 'twitter_user'@'%' IDENTIFIED BY 'كلمة_سر';
> GRANT ALL PRIVILEGES ON twitter_iraqi.* TO 'twitter_user'@'%';
> FLUSH PRIVILEGES;
> ```

### 3. شغّل السيرفر

**الطريقة الأسهل (سيرفر PHP المدمج):**

```bash
cd public
php -S localhost:8000
```

افتح المتصفح على: `http://localhost:8000`

**أو بـ XAMPP/Apache:**

خلي مجلد المشروع بمجلد `htdocs`، وخلي الـ Document Root يوجه إلى مجلد `public/`.

### 4. سجّل دخولك أو سوي حساب جديد

- تگدر تدخل بالحساب التجريبي: `ali` / `123456`
- أو تسوي حساب جديد من زر "أنشئ حساب جديد"

## هيكلية المشروع

```
twitter-clone/
├── app/
│   ├── Controllers/    → منطق التحكم (Auth, Post, Comment, Like, Follow...)
│   ├── Models/         → التعامل مع قاعدة البيانات (User, Post, Comment...)
│   ├── Services/       → خدمات مساعدة (رفع الصور)
│   └── Helpers/        → دوال عامة (auth, router, csrf...)
├── database/
│   └── database.sql    → السكيما الكاملة + بيانات تجريبية
├── routes/
│   └── web.php         → كل الروابط (Routes)
├── resources/
│   ├── views/           → صفحات HTML/PHP
│   ├── css/style.css    → التصميم
│   └── js/app.js        → اللايك بالـ AJAX
├── public/
│   ├── index.php        → نقطة الدخول (Front Controller)
│   └── uploads/          → الصور المرفوعة
└── .env                  → إعدادات قاعدة البيانات
```

## ملاحظات تقنية

- المشروع مكتوب بـ **PHP خام** (Vanilla PHP) بدون Composer أو Laravel، عشان يكون سهل التشغيل بأي سيرفر بدون تنصيب مكتبات.
- في راوتر بسيط (`app/Helpers/Router.php`) يتعامل مع الروابط المرنة زي `/profile/{username}`.
- كل الاستعلامات تستخدم **Prepared Statements** (PDO) لمنع SQL Injection.
- كلمات السر مشفّرة بـ `password_hash()` (bcrypt).
- الصور تنحفظ بـ `public/uploads/` وتترفع بحد أقصى 5 ميغا.
